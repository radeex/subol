The Subol Language

Compilation

In this order:

  • Identifier: Turned into a name lookup operation.
  • Number/String: Turned into a Const.
  • NewLine: ignored.
  • List
    • If the first element is an identifier naming a special form, the special is executed.
    • If the first element is an identifier naming a macro, the macro is executed.
    • Otherwise, look up the first identifier as a name and call it as a function.

Syntax

Code Lists

( )

(foo bar baz) creates a list to be evaluated. See `Compilation' above to see how.

Data Lists

[ ]

[1 2 3] creates a list with three elements, 1, 2, and 3.

Internally, it is translated to (mklist 1 2 3)

Blocks

It is possible to use indentation to delimit lists.

foo:
    bar

Is equivalent to (foo bar). That is, when a colon is followed by a newline and an indentation, an OpenParen is placed at the previous newline, and a CloseParen is placed at the un-indent that matches said indentation.

It is recommended to use this syntax for the following forms: if, elif, else, for, while, def, class, defmacro. For example:

class MyClass ():

    def (greet self):
        (println "`Hello world!' from" self)

    def (bye self):
        (println "`Good bye!' from" self)
